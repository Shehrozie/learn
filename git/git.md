# Git Notes

```bash
# To initialize git repository
git init

# Add changed files to repository, . means add all
git add .

# Add files to stage locally
git commit -m "First commit"

# Change the name of the master branch to main
git branch -M main

# Remote url
git remote add origin https://github.com/Shehrozie/learn.git

# PUSH to github
git push -u origin main

```
